# 项目概述

## 项目简介

Japan RWA (Real World Asset) 是一个基于区块链的房地产代币化平台,旨在将日本房地产资产转化为可交易的代币。

## 项目目标

1. 资产代币化
   - 将房地产资产转化为可交易的代币
   - 提供标准化的资产表示
   - 支持部分所有权

2. 交易便利性
   - 降低交易门槛
   - 提高流动性
   - 简化交易流程

3. 合规性
   - 符合日本法律法规
   - 确保交易安全
   - 保护投资者权益

## 核心功能

### 1. 房产管理

- 房产注册
- 房产审批
- 房产信息管理
- 房产状态追踪

### 2. 代币管理

- 代币创建
- 代币发行
- 代币转账
- 代币销毁

### 3. 用户管理

- 用户注册
- 权限管理
- 白名单管理
- 用户操作记录

## 技术架构

### 1. 智能合约

- PropertyRegistry: 房产注册合约
- TokenFactory: 代币工厂合约
- RealEstateToken: 房地产代币合约

### 2. 后端服务

- 配置管理
- 合约交互
- 事件处理
- 数据存储

### 3. 前端应用

- 用户界面
- 交易界面
- 资产管理
- 数据展示

├── frontend-tests/      # 前端测试

│   └── utils/         # 工具类
