# 日本房产通证化系统用户手册

## 1. 系统概述

日本房产通证化系统是一个基于区块链的平台，将日本实物房产转化为数字通证，实现房产所有权的分割、流通和管理。本系统部署在BSC网络上，用户可以通过支持Web3的钱包（如MetaMask）与系统交互。

## 2. 用户角色

### 2.1 超级管理员
系统的最高权限角色，负责系统管理、角色分配和合约升级等操作。

### 2.2 房产管理员
负责房产注册、审核和管理的角色。

### 2.3 普通用户
可以购买、持有和交易房产代币的用户。

## 3. 管理员操作指南

### 3.1 超级管理员操作

#### 3.1.1 角色管理
1. 打开系统管理界面
2. 选择"角色管理"功能
3. 输入用户地址和要授予的角色
4. 点击"授予角色"按钮
5. 确认交易并等待上链

#### 3.1.2 系统状态设置
1. 打开系统管理界面
2. 选择"系统状态"功能
3. 选择"活跃"或"非活跃"状态
4. 点击"更新状态"按钮
5. 确认交易并等待上链

#### 3.1.3 合约升级
1. 打开系统管理界面
2. 选择"合约升级"功能
3. 选择要升级的合约
4. 输入新实现合约的地址
5. 点击"升级"按钮
6. 确认交易并等待上链

#### 3.1.4 费用设置
1. 打开费用管理界面
2. 选择要修改的费用类型
3. 输入新的费用比例（万分比）
4. 点击"更新费用"按钮
5. 确认交易并等待上链

### 3.2 房产管理员操作

#### 3.2.1 注册房产
1. 打开房产管理界面
2. 选择"注册新房产"功能
3. 填写房产信息：
   - 房产ID
   - 国家
   - 元数据URI（包含详细信息的IPFS链接）
4. 点击"注册"按钮
5. 确认交易并等待上链

#### 3.2.2 创建代币
1. 打开房产管理界面
2. 选择已审核通过的房产
3. 点击"创建代币"按钮
4. 填写代币信息：
   - 代币名称
   - 代币符号
   - 初始供应量
5. 点击"创建"按钮
6. 确认交易并等待上链

#### 3.2.3 创建租金分配
1. 打开租金管理界面
2. 选择"创建新分配"功能
3. 填写分配信息：
   - 房产ID
   - 代币地址
   - 稳定币地址
   - 租金总额
4. 点击"创建"按钮
5. 确认交易并等待上链

#### 3.2.4 发起赎回流程
1. 打开赎回管理界面
2. 选择"发起赎回"功能
3. 填写赎回信息：
   - 房产ID
   - 代币地址
   - 稳定币地址
   - 赎回总额
4. 点击"发起"按钮
5. 确认交易并等待上链

## 4. 普通用户操作指南

### 4.1 购买代币

#### 4.1.1 在二级市场购买
1. 打开市场界面
2. 浏览可用的销售订单
3. 选择要购买的订单
4. 点击"购买"按钮
5. 确认交易并等待上链

### 4.2 出售代币

#### 4.2.1 创建销售订单
1. 打开市场界面
2. 选择"创建订单"功能
3. 填写订单信息：
   - 代币地址
   - 代币数量
   - 价格
   - 稳定币地址
4. 点击"创建"按钮
5. 确认交易并等待上链

#### 4.2.2 取消销售订单
1. 打开市场界面
2. 找到自己创建的订单
3. 点击"取消"按钮
4. 确认交易并等待上链

### 4.3 领取租金

#### 4.3.1 查看可领取的租金
1. 打开租金界面
2. 系统显示可领取的租金分配列表
3. 查看每个分配的详细信息

#### 4.3.2 领取租金
1. 打开租金界面
2. 选择要领取的租金分配
3. 点击"领取"按钮
4. 确认交易并等待上链

### 4.4 赎回代币

#### 4.4.1 申请赎回
1. 打开赎回界面
2. 选择处于赎回状态的房产
3. 输入要赎回的代币数量
4. 点击"申请赎回"按钮
5. 确认交易并等待上链

#### 4.4.2 查看赎回状态
1. 打开赎回界面
2. 查看自己的赎回申请列表
3. 查看每个申请的状态

## 5. 常见问题解答

### 5.1 如何连接钱包？
1. 安装MetaMask或其他支持BSC网络的钱包
2. 添加BSC网络配置
3. 在系统界面点击"连接钱包"按钮
4. 选择您的钱包并授权连接

### 5.2 为什么我无法转移代币？
代币转移可能受到以下限制：
- 房产处于赎回状态
- 房产已下架
- 系统处于非活跃状态
- 您的账户被暂停

### 5.3 如何查看我的代币余额？
1. 连接钱包后，系统会自动显示您的代币余额
2. 您也可以在"我的资产"页面查看详细信息

### 5.4 租金如何计算？
租金按照您持有的代币比例计算：
- 应得租金 = 净租金 × (您的代币数量 / 代币总供应量)
- 净租金 = 租金总额 - 平台费用 - 维护费用

### 5.5 赎回流程需要多长时间？
赎回流程通常包括以下步骤：
1. 管理员发起赎回流程
2. 代币持有者申请赎回
3. 管理员审核并处理赎回
整个过程可能需要几天到几周不等，具体取决于项目方的处理速度。
